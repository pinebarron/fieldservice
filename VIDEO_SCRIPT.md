## RECORDING TIPS

- Use actual app with realistic test data (not empty screens)
- Create a fictional solar company "SunPro Installs" for the demo
- Have 3-4 jobs scheduled for "today"
- Include real-looking customer names and addresses
- Show the mobile view on an actual phone (screen recording)
- Keep transitions smooth — no awkward loading screens
- Background music: upbeat but professional, not distracting
